# Estudio de notas de pregrado FACSO UChile

Este repositorio contiene el estudio de las notas de pregrado de la Facultad de Ciencias Sociales de la Universidad de Chile (FACSO UChile). 

El estudio estuvo a cargo de [DATASOC](https://datasoc.cl/), centro de datos sociales de FACSO.

> [!NOTE]  
> La estructura del repositorio es la siguiente (basada en el protocolo IPO):

```
├── input: Información de entrada para el desarrollo de los análisis:
│ ├─── data: Bases de datos.
| | ├─── raw_data: Bases en bruto (no disponibles por anonimización).
| | ├─── proc_data: Bases procesadas. 
│ ├─── documents: Documentos técnicos que orientan el trabajo con los datos sin procesar.
│ ├─── libs: Bibliografía y diseño customizado (CSS)
| ├─── images: Imágenes.
│
├── proc: Scripts donde se procesaron datos.
│
├── output: Archivos generados a partir de los análisis (en formato QMD y HTML).
│ ├─── presentations: Presentaciones para la difusión de resultados.
│ ├─── reports: Reportes de investigación.
|
|── docs: Informe final renderizado.
```
